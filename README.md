# Single-Cell RNA-seq — Interactive Tutorial

**🌐 Live demo / 線上瀏覽**: <https://charlene717.github.io/scrna-interactive-tutorial/>

## 中文簡介

單細胞 RNA-seq (scRNA-seq) 互動式教學。涵蓋 QC、PCA、UMAP、分群、軌跡推論與 CellChat 等核心分析流程，繁體中文 / 英文雙語並陳，瀏覽器直接開啟 `index.html` 即可使用。

## English Description

A bilingual (繁體中文 / English) interactive scRNA-seq tutorial covering QC, PCA, UMAP, clustering, trajectory inference, and CellChat — open `index.html` in any modern browser to begin. No build step, no server required.
